# 마크 다운 작성법
1. 제목 
    * (# 사용)
    * 개수에 따라 크기 감소
2. 목록
    * 순서는 1,2,3 사용
    * 순서 없는 것은 -, +, * 사용 
3. 강조
    * 기울임 *글자*
    * 굵게 **글자**
    * 취소 ~~글자~~
4. 코드 
    * 인라인 코드:(백틱 사용) `print(Hello world)`
    * 블록 코드:(백틱 세번) ``` for (i>0): print i```
5. 링크
    * []()동시 사용
    * [GOOGLE](https://google.com)
6. 이미지 
    * ![]()사용
7. 인용
    * 주석이나 인용문구 표현
    * '>' 를 사용 중첩 가능
8. 표 
    * 인터넷에 마크다운 표 사용
9. 수평선
    * 구분선 -, *, _ 중 하나 연속 세번 사용
