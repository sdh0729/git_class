# 깃허브 연결 커맨드
* `git remote add origin  주소` - 레포지토리와 연결
* `git remote -v` -조회
* `git remote rm origin` -삭제
* `git push origin master` - 깃허브에 올리기

# 클론과 풀
* `git clone 주소` 깃 init 과 remote 는 이미 수행 됨
* `git pull origin master`
